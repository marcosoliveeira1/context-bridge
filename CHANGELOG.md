
# Change Log

All notable changes to the "context-bridge" extension will be documented in this file.

## [1.0.0] - 2026-03-10

### Added

* Initial release under the name **Context Bridge**.
* Support for project-wide configuration via `.compiladorai`.
* Sidebar UI with persistent ignore list management.
* Path Traversal security check during the "Spread" process.
* Multi-file extraction from single text input.
* Project-to-text compilation for LLM context.
